== Installation ==

1. Upload the `hippoo-notification` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Hippoo Notification settings page and configure the plugin.

== Frequently Asked Questions ==

= How do I configure the plugin? =

To configure the plugin, go to the Hippoo Notification settings page and enter your OneSignal App ID and API Key.

= Can I customize the notification message? =

Yes, you can customize the notification message by editing the message template in the Hippo app notification.
